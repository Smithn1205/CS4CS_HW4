Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS), is a stylesheet language, which is mainly used for HTML and XML. It is used to enhance the webpages, like editing fonts, colors, positioning and enhancing user experience.

Key points:

1. CSS is used to make the webpage visually appealing.
2. It saves time, as we can link various webpage to one single css file.
3. It is compatible with laptops as well as smartphones.
4. You can easy make any changes to the file.
5. It is much organized and systematic.
